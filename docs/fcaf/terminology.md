# Terminlogy

Terminology builds on terminology used in the [ARF]. See annex I of the ARF, for definitions of terminology.

## Terminology mapping

Test case specifications relate to both the EUDI-wallet as well as underlying Standards and Technical Specifications. Standards and their governing Standards Development Organization (SDO) often use other terminology than those used in the ARF.
The following table is provided to support testing EUDI-wallet requirements to standards and vice versa. It is provided for convenience as best effort. This may contain inaccuracies and is non-exhautive, use at own risk.

| ARF Term | ISO mdoc term |
| --- | --- |
| Attestation | mobile document, mdoc |
| Attribute | data element |
| Attestation Provider | issuing authority, issuer |
| Relying Party instance | mdoc reader |
| Relying Party | mdoc verifier |
| Wallet instance | mdoc app |
| WSCA/WSCD | secure area |
| (Wallet-relying party) access certificate | reader authentication certificate |
| Wallet Provider | mdoc app provider |
